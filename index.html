YOLKALYZER

A Collada DAE "glitcher".

A Collada DAE "glitcher" / "texturizer".

Inspired by Satromizer and can be considered as such:

BITMAP => Satromizer :: 3D => YOLKALYZER

USAGE:

  1. Clone the repo git clone git://github.com/yyolk/yolkalyzer.git

  2. Change into the directory and install dependencies with bundle.

    cd yolkalyzer
bundle install

  3. Run the script. ruby yolkalyzer.rb mesh/model.dae mesh/fuckedupmodel.dae
